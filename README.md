# StressCam

> Contactless Human Stress Detection using Computer Vision, rPPG and Machine Learning.

---

## Project Goal

Develop a modular pipeline for contactless stress assessment from facial videos using:

- rPPG
- Facial Features
- Physiological Feature Extraction
- Machine Learning
- Deep Learning
